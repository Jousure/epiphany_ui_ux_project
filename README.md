**Epiphany** is a web-based UI/UX project designed to connect readers and writers, providing an intuitive platform for discovering, reading, and publishing novels. The project focuses on accessibility, engagement, and seamless navigation.

## Key Features
- **Flexible Reading Access**: Credit-based system and premium subscription  
- **Story Discovery**: Genre filters, recommendations, and bookmarks  
- **Writer Dashboard**: Manage stories, edit chapters, and track analytics  
- **Milestone-Based Monetization**: Rewards for writers as their stories grow  
- **Clean & Responsive UI**: Dark/light mode, reading progress tracking  

## Design Highlights
- High-fidelity mockups created in **Figma**  
- Consistent color palette and typography (Primary: #7803AA, Secondary: #B078C8; Headings: Merienda, Body: Roboto)  
- User flows designed for both readers and writers  
- Accessibility-focused layouts for readability and responsiveness  

## User Personas
- **Reader Persona**: Wants to explore novels easily, track reading progress, and access content flexibly  
- **Writer Persona**: Seeks visibility, analytics, and monetization for published stories  

---

This project demonstrates **UI/UX design, wireframing, high-fidelity mockups, user flows, and accessibility-focused design** for a modern digital reading and writing platform.
